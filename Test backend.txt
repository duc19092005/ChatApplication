REQUIREMENT:
websocketd
pnpm
cd backend/
pnpm start
REGISTER:
curl -X POST http://localhost:8080/api/auth/register -H "Content-Type: application/json" -d '{"username":"user","password":"user"}'

LOGIN:
curl -X POST http://localhost:8080/api/auth/login -H "Content-Type: application/json" -d '{"username":"testuser","password":"testpass"}'
-output: token...
MESSAGES:
curl -X GET http://localhost:8080/api/chat/messages -H "Authorization: Bearer <user_token>"

WEBSOCKET:
wscat -c "ws://localhost:8080?token=<user_token>"
